## 🧠 What I Learned

- Creating and inspecting DataFrames from dictionaries, lists, and files
- Indexing and selecting data with `.loc[]` and `.iloc[]`
- Sorting DataFrames with `.sort_values()`
- Identifying and handling missing data with `.isna()` and `.dropna()`
- Removing duplicates with `.drop_duplicates()`
- Fixing inconsistent values with `.replace()` and string methods
- Combining multiple cleaning steps into a full data-cleaning workflow
